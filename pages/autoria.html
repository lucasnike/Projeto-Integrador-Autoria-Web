3 AUTORIA WEB

3.1 Hospedagem

O site foi hospedado no Github Pages que é um host de sites gratuito, que tem o objetivo de facilitar o processo difícil e cansativo que envolve a hospedagem de um site em outros hosts convencionais. No github pages é bem simples de hospedar, basta criar um repositório no Github, colocar o código Html, CSS e JavaScript, e depois basta ir nas configurações desse repositório e o associar à uma página do Github Pages e o deploy estará feito

3.2 Layout

O site foi criado com HTLM5 , CSS3, JavaScript e o framework bootstrap. Inicialmente o site foi desenvolvido com a linguagem de marcação (HTML5) e estilizado (CSS3), foi utilizado bootstrap para a criação do layout e javaScript para as animações.

3.3 Responsidade

Caso o site seja aberto em um dispositivo mobile, ele se adapta para uma interface mais simples. Para a criação do sidebar foi usada a tag nav e possuindo nela a tag ul (que é uma tag de lista não ordenada) a mesma contém uma tag que faz referência a outros arquivos.
