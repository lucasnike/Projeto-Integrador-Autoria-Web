4 ORGANIZAÇÃO E MANUTENÇÃO DE COMPUTADORES

4.1 INTRODUÇÃO AO CAPÍTULO

Neste capítulo trabalhamos a especificação de equipamentos de microinformática, sendo este um dos conteúdos da base científico-tecnológica da disciplina de Organização e Manutenção de Computadores.

O Projeto Integrador tem como título “Especificação de um ambiente virtual voltado para a aprendizagem dos alunos”, dentro desta proposta e do conteúdo da disciplina vamos especificar computadores que possam executar programas essenciais para um bom desempenho durante o curso de Informática. Alguns desses softwares são: Portugol Studio, Eclipse, NetBeans, Visual Studio Code, Sublime Text, MySQL Workbench.

Para que todo o ambiente funciona perfeitamente estamos especificando a configuração mínima de pelo menos três tipos de computadores que são usados em cenários diferentes, como: disponibilização de laboratórios de informática para acesso aos alunos que estão sem o hardware mínimo recomendado, uso por parte dos alunos de computadores pessoais em suas residências e uso por parte dos professores de computadores pessoais em suas residências. Abaixo seguem os três casos citados:

  1. Configuração mínima dos computadores do tipo desktop para os laboratórios. (Computador 01);
  2. Configuração mínima dos computadores pessoais dos alunos para acessarem o sistema de forma remotamente. (Computadores 02);
  3. Configuração mínima dos computadores pessoais dos professores para gravarem aulas de forma remota. (Computadores 03);

4.2 CONFIGURAÇÃO MÍNIMA DOS COMPUTADORES PARA OS LABORATÓRIOS (COMPUTADOR 01)

Nesta seção apresentaremos a configuração mínima para os computadores que serão utilizados nos laboratórios. Estes computadores devem ser do tipo Computadores de Rede, que são computadores mais robustos, visto que passam mais tempo ligados diariamente. As especificações seguem o padrão utilizado para licitações do setor público, logo modelos, versões ou especificações diretas não são aceitas.

4.2.1 - Placa principal (placa mãe)

Suporte a dual channel no barramento da memória; Suporta o padrão ECC para memória RAM Possui 11 (onze) portas USB, sendo 4 USB 3.1 + 1 x USB 3.1 Type C + 2x Thunderbolt (USB Type-C) sendo estas 2 utilizadas através dos slots PCIe disponíveis na placa mãe, frontais; e 2 USB 3.1 + 2 USB 2.0 traseiras. Possui 03 interfaces SATA 3.0; Arquitetura ATX ou BTX.

4.2.3 - Interfaces

4.2.3.1 - Interface de vídeo Compatibilidade com DirectX 12 ou superior e OpenGL 4.5 ou superior; 02 (duas) saídas digitais (DisplayPort ou HDMI). No caso de duas saídas digitais idênticas, deve ser fornecido pelo menos um adaptador para o outro padrão. Deve ser fornecido pelo menos um adaptador da saída digital para DVI; A placa de vídeo deverá ser integrada ao processador.

4.2.3.2 - Interface de som de no mínimo 16 bits; Alto-falante interno integrado; Entrada na parte frontal do gabinete combinada para microfone/fone de ouvido (combinada ou individuais).

4.2.3.3 - Interface de rede Gigabit Ethernet (RJ-45). Configuração totalmente por software; velocidade de 100/1000 Mbits; 1.6.3. Full duplex; Padrão IEEE 802.3 com tecnologia WOL (Wake on LAN); 1.6.5. Integrada à placa-mãe.

4.2.4 - Processador

Processador 2246 G Possui índice CPU Benchmark Passmark de 12.000 pontos ou superior; seis ou mais núcleos físicos, suportando pelo menos 12 threads de processamento; Possui TDP de até 140W; Suporte a conjunto de instruções 64 bit; Suporte a virtualização de processamento (VT-x ou equivalente) e de I/O (VT-d ou equivalente); Com cooler original do mesmo fabricante do processador (certificado pelo fabricante do processador), especificado pelo fabricante para o modelo do processador ou sistema de ventilação original do fabricante do equipamento capaz de manter o processador e todos os periféricos em perfeito funcionamento.

4.2.5 - Memória Primária

Memória 16 (dezesseis) GB de DDR4, em configuração dual-channel (pentes em pares) no padrão ECC; Memória expansível até pelo menos 64 GB sem remover a memória já fornecida; Velocidade padrão DDR4-2400Mhz ou superior; Suporte a pelo menos 4 (quatro) slots de memória.

4.2.7 - Gabinete

Tipo tool less, que permite abertura do gabinete e remoção das unidades de armazenamento sem o uso de ferramentas. Será aceito o uso de um parafuso exclusivamente para a fixação do disco rígido SSD, caso esse seja ofertado no padrão M.2; Sistema de Segurança incluso no chassi do equipamento, sem uso de adaptadores e/ou a possibilidade de adicionar cadeado em local específico original de fábrica para evitar acesso ao interior do gabinete; Sistema de detecção de intrusão de chassis.

4.2.8 - Teclado

Teclado com bloco numérico separado; Com Layout Português Brasil (ABNT2); Com ajuste de inclinação; conectado por cabo USB ao computador.

4.2.9 - Mouse

Mouse óptico; Tamanho padrão (não mini-mouse); Com pelo menos 2 botões mais botão de rolagem; Modelo óptico laser; Conectado por cabo USB ao computador; Resolução de pelo menos 1200 dpi; Formato ergonômico ambidestro; Com mouse pad adequado ao modelo do mouse.

4.2.10 - Energia

Fonte com chaveamento automático, suportando as tensões de entrada de 110v/220v, integrada (interna); PFC ativo; Eficiência mínima de 85% a 110v, sob qualquer carga entre a faixa de 20% a 100%.
